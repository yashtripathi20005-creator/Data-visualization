# 📊 Data Visualization Dashboard

A Flask-based data visualization dashboard using Chart.js with real-time analytics and interactive charts.

## ✨ Features
- 4 interactive chart types (Line, Bar, Doughnut)
- Real-time data refresh
- Responsive design
- REST API endpoints
- Auto-generated sample data

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
